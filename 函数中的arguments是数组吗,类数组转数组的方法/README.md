### 函数中的arguments是数组吗,类数组转数组的方法

1. 不是
2. 列数组转成`array`
	- `[...arguments]` ...运算符
	- `Array.from(arguments)`
	- `Array.prototype.slice.call(arguments)`
	- 通过循环`for in` 或其他 Object 循环 把值**塞到**新数组中

#### 类数组
- 拥有length属性，其它属性（索引）为非负整数(对象中的索引会被当做字符串来处理，这里你可以当做是个非负整数串来理解）
- 不具有数组所具有的方法
